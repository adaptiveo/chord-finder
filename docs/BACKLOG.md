## P1 — Важно (в течение месяца)

### BACK-001: Автодополнение при вводе аккорда
**Статус:** [ ] В плане / [ ] В работе / [x] Реализовано / [ ] Отменено
**Дата реализации:** 2026-04-22
**ADR:** [ADR-002-autocomplete-dropdown](adr/ADR-002-autocomplete-dropdown.md)

**Реализация:**
- Dropdown с подсказками появляется при вводе первых букв
- Поддерживает все аккорды из базы (включая Gsus4)
- Выбор по клику автоматически заполняет поле ввода
- Исправлен баг с повторным открытием dropdown (ERR-009)

**Известные баги:**
- ERR-009: Dropdown открывался повторно после выбора — исправлен через флаг `justSelected`

**Проблема:** Пользователь вводит название аккорда вручную, не зная точного формата

**Предложение:** Добавить dropdown с подсказками по мере ввода: ввёл "C" → показываем C, C7, Cm, Cmaj7, Cadd9...

**Ожидаемый результат:** Быстрый выбор без полного набора, открывает пользователю существующие варианты

**Теги:** #ux #search #autocomplete

### BACK-016: Автоопределение раскладки клавиатуры
**Статус:** [ ] В плане / [ ] В работе / [ ] Реализовано / [ ] Отменено

**Проблема:** Пользователь вводит "С" (русская раскладка) вместо "C" (английская), аккорд не находится

**Предложение:** Автоматически транслитерировать русские буквы в английские эквиваленты при нормализации:
- С → C, с → c
- А → A, а → a
- В → B (русская В = английская B)
- Е → E, е → e
- И т.д.

**Примеры:**
| Ввод (рус.) | Конвертация | Результат |
|-------------|-------------|-----------|
| См | Cm | C minor |
| Ам | Am | A minor |
| С7 | C7 | C dominant 7 |
| Фа | F | F major |

**Ожидаемый результат:** Пользователь может забыть переключить раскладку — приложение всё равно найдёт аккорд

**Теги:** #ux #input #i18n #cyrillic

**Проблема:** Нет быстрого доступа к часто используемым аккордам

**Предложение:** Сохранять последние 10 просмотренных + возможность добавить в избранное (звёздочка)

**Ожидаемый результат:** Горизонтальный список под поиском: "Недавние: C Am F G"

**Теги:** #ux #history #favorites

## Workflow: Бэклог → ADR

Когда фича из бэклога реализуется:

1. **Обновить BACKLOG.md** — изменить статус фичи:
   ```markdown
   **Статус:** [x] Реализовано
   **Дата реализации:** 2026-04-22
   **ADR:** [ADR-005-material3-dynamic-colors](adr/ADR-005-material3-dynamic-colors.md)
   ```

2. **Создать ADR** — задокументировать архитектурное решение

3. **Обновить PROJECT.md** — добавить ссылку на ADR в разделе архитектурных решений

> Фича остаётся в бэклоге в своём приоритетном разделе — это история развития проекта.
