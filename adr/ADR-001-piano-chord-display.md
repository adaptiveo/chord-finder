# ADR-001: Отображение аккордов для пианино

**Дата:** 2026-04-22
**Статус:** ✅ Принято и частично реализовано
**Контекст:** #bug #ui #piano
**Автор:** @user

## Контекст

Пользователь заметил отличия в отображении аккордов для пианино между приложением и примерами в интернете. Возможные проблемы:

1. Ноты отображаются без подписей (не видно, какая нота C, D, E и т.д.)
2. Октавы могут отображаться неправильно (слишком высоко/низко)
3. Расположение черных клавиш может не соответствовать стандартному пианино

## Текущая реализация

### Данные аккордов
Ноты хранятся как номера полутонов от C:
- C = 0, C# = 1, D = 2, D# = 3, E = 4, F = 5
- F# = 6, G = 7, G# = 8, A = 9, A# = 10, B = 11

Пример для C major: `Fret(1, 0), Fret(2, 4), Fret(3, 7)` → C, E, G

### Отображение
- 14 белых клавиш (2 октавы)
- Черные клавиши позиционируются между белыми
- Ноты аккорда подсвечиваются цветом

## Рассмотренные варианты

### Вариант 1: Добавить подписи нот на клавишах
- **Плюсы:** Понятно, какая нота где находится
- **Минусы:** Может быть мелко на маленьких экранах

### Вариант 2: Использовать только одну октаву с центральной C
- **Плюсы:** Стандартное отображение в учебниках
- **Минусы:** Не все аккорды поместятся

### Вариант 3: Добавить отображение нот текста (C, E, G) рядом с клавиатурой
- **Плюсы:** Чётко видно состав аккорда
- **Минусы:** Требует дополнительного места

### Вариант 4: Показывать несколько вариантов расположения аккорда
- **Плюсы:** Как в реальных учебниках по пианино
- **Минусы:** Сложнее реализация

## Решение

Предлагается комбинированный подход:

1. **Добавить текстовое отображение нот** (C, E, G) над клавиатурой
2. **Добавить подписи на белых клавишах** (C, D, E, F, G, A, B)
3. **Подсветить соответствующие октавы** для каждой ноты
4. **Добавить варианты** (root position, 1st inversion, 2nd inversion)

## Последствия

### Положительные
- Соответствие стандартным учебникам по пианино
- Понятнее для пользователей

### Отрицательные
- Требует изменения UI и данных
- Нужно добавить инверсии аккордов

## Связанные файлы
- `ChordDiagram.kt` - отображение клавиатуры
- `ChordData.kt` - данные аккордов
